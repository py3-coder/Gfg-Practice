<p><span style="font-size:18px"><strong>Example:</strong></span></p>

<pre><span style="font-size:18px"><strong>Input:
</strong>5<strong>
Output:
</strong>0 1 1 2 3 5</span>
<span style="font-size:18px"><strong>Explanation:</strong>
